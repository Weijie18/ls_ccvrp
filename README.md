# Repository File Description

This repository contains the source code, benchmark instances, vehicle-number settings, and prompt document used for solving the CCVRP.

## Files

* CCVRP.cpp
  The main C++ source file. It contains the `main` function and the implementation of the algorithm.

* Instance.zip
  The benchmark instances used for testing.

* Instance Vehicle Num.zip
  The corresponding number of available vehicles for each benchmark instance.

* prompt.md
  The integrated and optimized prompt document.

## Running Notes

When running `CCVRP.cpp`, the instances in `Instance.zip` and the corresponding vehicle numbers in `Instance Vehicle Num.zip` are used as input. The input format and file paths are specified in the C++ code.

Before running the code on a local machine, please update the input paths according to your own directory structure. The output file path should also be adjusted as needed based on your local configuration.
